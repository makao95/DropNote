# DropNote
Simple cross-platform note taking app created with Qt and QML. Ideal for journal and text-based notes. Some of it's features:
- themes (including dark one)
- local database (SQLite)
- notebooks
- UI in languages: english, polish
- calendar

In plans:
- tags
- Dropbox sync
- Encryption of notes 
- Mobile and web
